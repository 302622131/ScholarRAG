---
name: statistics-report
description: 统计知识库中某个研究方向的论文数据，生成统计报告
---

**你必须严格按以下步骤执行，不得跳过任何一步。**

## 步骤 1 — 确认统计维度

向用户提问，确认要统计什么：
- 统计哪个研究方向或主题？
- 有没有特别关注的维度（年份分布、页数、研究热度变化）？

在用户明确回答之前，不要进行搜索。

## 步骤 2 — 搜索论文并收集元数据

根据用户回答，调用 search_papers(top_k=20) 搜索相关论文。

将每篇论文的以下信息整理成一个 JSON 数组，写入 skill 目录下的 `_data.json`：
- paper_id: 论文 ID
- title: 标题
- year: 从内容推断年份（如"2023"、"2021"），如果无法推断填 0
- pages: 从内容推断的大致页数，如果无法推断填 0

JSON 格式示例：
```json
[
  {"paper_id": "abc123", "title": "...", "year": 2023, "pages": 12},
  ...
]
```

用 Python 代码写入文件（调用 run_python 执行一个简单的写文件脚本，或直接用 shell）。

## 步骤 3 — 执行统计脚本

调用 run_python 执行 analyze.py，传入参数：
- skill_name: "statistics-report"
- script: "analyze.py"
- args: ["_data.json"]

## 步骤 4 — 撰写统计报告

根据脚本输出的 JSON 结果，撰写报告：

**概览** — 共多少篇论文，覆盖什么时间段。

**年份分布** — 哪几年论文最多，趋势如何（上升/平稳/下降）。

**发表热度** — 该方向的论文数量变化趋势，可以推断的研究热度。

**其他发现** — 从数据中观察到任何值得注意的模式。

## 格式要求

- 自然段落，段间空行分隔
- 数字用简单编号呈现
- 不用 emoji，不用 markdown 标题符号
- 中文撰写
